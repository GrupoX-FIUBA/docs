Arquitectura

En esta sección desarrollaremos el modelo de la arquitectura pensada para este proyecto.

📄️ Modelo

Para el desarrollo de la aplicación adoptamos el siguiente modelo de servicios:

📄️ Backend

Kubernetes, microservices, API Gateway

📄️ Backoffice

Para el desarrollo del backoffice, se utilzó React como creador de sitios web estáticos. El mismo corre en Github Pages, haciendo todos los requests necesarios del lado del cliente mediante consultas diferidas. Para el diseño, se eligió la biblioteca de CSS Tailwind.

📄️ Android APP

La aplicación para móviles con Android se desarrolló utilizando React Native y Expo. Para mantener los estados se utilizó Redux. Además, se implementó un sistema de internacionalización con el estándar i18n.
